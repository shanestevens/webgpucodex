# WebGPU Codex

An interactive Three.js + WebGPU + TypeScript learning gallery that moves from basic geometry to lighting, shaders, compute, skinning, terrain, and advanced GPU labs.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## GitHub Pages

This repo includes a GitHub Actions workflow that builds and deploys the site to GitHub Pages whenever `main` is updated.

Expected Pages URL:

`https://shanestevens.github.io/webgpucodex/`
